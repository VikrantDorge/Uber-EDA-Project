# 🚗 Uber Ride Data Analysis (EDA)

## 📊 Project Overview
This project analyzes Uber ride data to understand patterns in ride demand based on time and location.

## 🔍 Key Analysis
- Peak hours analysis
- Day-wise ride trends
- Monthly demand patterns
- Base-wise ride distribution

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 📈 Insights
- Ride demand is highest during peak hours
- Weekends show higher activity
- Certain bases handle more rides

## 📁 Dataset
Dataset sourced from Kaggle (Uber Ride Data)
